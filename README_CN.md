# Arduino Core for RoboMaster i.MX RT Boards

本项目为 NXP I.MXRT 系列芯片开发板提供了 Arduino 兼容层，支持：

- 机甲大师通用拓展模块

获取更多资料，请参考在线帮助文档

[RoboMaster扩展模块 用户指南](https://rm-edu-boards.readthedocs.io/)
